# MouseMover 

A lightweight Python script that keeps your system or app active by periodically moving the mouse a tiny amount.
Perfect for preventing AFK timeouts or idle disconnects when allowed.

---

# Features
- Periodically nudges the mouse (right then left) every 5 minutes
- Hotkeys while running:
  - **P** — Pause/Resume
  - **Q** — Quit
- Minimal, readable Python code
